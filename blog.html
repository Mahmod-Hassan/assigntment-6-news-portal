Commont Interview Questions

1. What is the difference among var,let and const ?

var let const
var variables are function or global scoped let is block scoped const is block scoped
var variables can be updated and re-declared within its scope; let variables can be updated but not re-declared; const variables can neither be updated nor re-declared.
var can be declared without being initialized, let can be declared without being initialized, const must be initialized during declaration.

2. What is the difference between arrow function and regular function?

Arrow Function Regular Function
Arrow Function do not have their own this Regular Function have their own this
arrow functions provide an implicit return but the regular function don't

3. What id the difference among map, forEach, find, filter and reduce?

map(): map creates a new array by transforming every element in an array individually.

filter(): filter checks every element in an array to see if it meets a certain criteria and returns a new array with the elements that return truthy for the criteria.

forEach(): forEach is used to execute the same code on every element in an array but does not change the array and it returns undefined.

reduce(): reduce() method in JavaScript is used to reduce the array to a single value and executes a provided function for each value of the array (from left-to-right) and the return value of the function is stored in an accumulator.

find(): find method returns the value of the first element that passes a test. If no elements are found it return undefined
